Révision HTML, CSS et Bootstrap

Jour 1 : HTML

Structure de base :

<!DOCTYPE html>
<html>
<head>
    <title>Titre</title>
</head>
<body>
    <h1>Bonjour le monde !</h1>
</body>
</html>

Balises importantes :

  • <h1> à <h6> : Titres
  • <p> : Paragraphes
  • <a href="#"> : Liens
  • <img src="image.jpg" alt="description"> : Images
Astuce : Utilise les balises sémantiques comme <header>, <main>, et <footer> pour une meilleure structure.

Jour 2 : CSS

Sélecteurs :

p {
    color: blue;
    font-size: 16px;
}

Flexbox :

.container {
    display: flex;
    justify-content: center;
    align-items: center;
}
Exercice : Crée une mise en page avec Flexbox.

Jour 3 : Bootstrap

Grille :

<div class="container">
    <div class="row">
        <div class="col-md-6">Colonne 1</div>
        <div class="col-md-6">Colonne 2</div>
    </div>
</div>
Astuce : Utilise container-fluid pour une largeur totale.

Jour 4 : Projet pratique

Crée un site web simple en utilisant HTML, CSS et Bootstrap.
